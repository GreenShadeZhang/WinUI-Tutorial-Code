## WinUI-Tutorial-Code

### 主要是一些Windows App SDK（WinUI3） 学习记录教程的示例代码


## ***WinUI.UseLiteDB***

这个项目展示了winui结合liteDb进行数据的存储 对于大量的图片放在数据库文件里 比放在Assets目录里会流畅很多
